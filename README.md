# 📖 Overview

AI Career Mentor is an interactive web application that helps aspiring software engineers and AI enthusiasts evaluate their current technical skills and receive a personalized roadmap toward their desired career.

The application compares user skills against industry-relevant requirements and recommends:

* ✅ Missing Skills
* 🚀 Real-world Projects
* 📚 Learning Resources
* 📈 Career Readiness Score
* ⏳ Estimated Learning Time

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/riazaslam029/AI-Career-Mentor.git
```

Go inside the project

```bash
cd AI-Career-Mentor
```

Create Virtual Environment

```bash
python3 -m venv venv
```

Activate Virtual Environment

Linux/macOS

```bash
source venv/bin/activate
```

Windows

```bash
venv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run the Application

```bash
streamlit run app.py
```
